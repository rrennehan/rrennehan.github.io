Programming Assignments

Note: An up-to-date version of Java is required to run the jar files.

Object Oriented RPG Game (.jar file)
This Java project uses object orientated concepts such as abstraction, polymorphism, inheritance, and encapsulation to make a battle game. The user chooses one of three unique heroes to battle against a monster where a winner is determined. The user can upgrade stats, buy items, and advance to defeat increasingly difficult monsters.
(Alternatively click here to see the java files)
Animal Recorder (.jar file)
For this Java assignment, we made a program that would allow researches to record details about antarctic animals. This includes its body weight, sex, coordinates, and more for one of three animal species. It stores the details for each entry into an external txt file using file input and output methods. The project also demonstrates object oriented design. Each animal has its own class with its own properties. Each animal is a child of an animal class that hosts shared properties between each.
(Alternatively click here to see the java files)
Bouncing Shapes Screensaver (.jar file)
A screensaver-like application in Java. By clicking inside, a shape is generated. Shapes bounce off the walls and off each other. Makes successful use of late-binding, java.awt graphics, and a timer to update the screensaver 60 times per second.
(Alternatively click here to see the java files)
Countries of the World (.zip folder)
A graphical application made using PyQT5 designer. The user is presented with a list of all countries. When the user clicks on a country, more information is provided. To run it, download the zip folder, extract it to a new location, and run the CountriesApplication.exe file from the new location. The python code is also provided.
BlackJack Game (.zip folder)
A personal project I did using Python. I wanted to put all the introductory programming skills I learned in my introduction to programming course to the test.
