Programming Assignments

Object Oriented RPG Game (.jar file)
This Java project uses object orientated concepts such as abstraction, polymorphism, inheritance, and encapsulation to make a battle game. The user chooses one of three unique heroes to battle against a monster where a winner is determined. The user can upgrade stats, buy items, and advance to defeat increasingly difficult monsters. (Alternatively click here to see the java files)
Animal Recorder (.jar file)
For this Java assignment, we made a program that would allow researches to record details about antarctic animals. This includes its body weight, sex, coordinates, and more for one of three animal species. It stores the details for each entry into an external txt file using file input and output methods. The project also demonstrates object oriented design. Each animal has its own class with its own properties. Each animal is a child of an animal class that hosts shared properties between each. (Alternatively click here to see the java files)
BlackJack Game (.zip folder)
A personal project I did using Python. I wanted to put all the introductory programming skills I learned in my introduction to programming course to the test. To play it, simply download the zip folder above, extract all the files to a new location, and load the Blackjack.exe application from the new location. The python file is also included so you can view the code. Comes with a functioning leaderboard.
